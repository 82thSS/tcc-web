概要

この班では、主にゲームやソフトウェアのプログラミングをします。
PythonやC#を始め、C++やJavaScriptなど幅広い言語を扱っています。

プログラミング班は班員の数が3つの班の内最多で、
全学年の部員が所属しているので先輩達から色々な知識が学べます。


制作環境

部室のPCの性能としては、基本的にどんなゲームでも
製作できるほどのハイスペックなものが備えられています。

主に使用しているツールは
・Visual Studio Code : プログラムのコードを編集するソフト
・Unity : C#のゲームを簡単に作れるゲームエンジン
・Pygame : Pythonのゲームを簡単に作れるゲームエンジン
等ですが、部員の使用している言語やエンジンに応じて
順次追加していくつもりです。




桐朋祭では？

毎年6月に行われる桐朋の文化祭「桐朋祭」では、
例年数学教室にて部員が制作したゲームの展示を行っています。
毎年展示されている作品は異なるので、期間限定でプレイできる
コンテンツのようなものだと思ってもらえれば結構です。

毎年コンピューター部は来場者のアンケートで上位にランクイン
できていないのでお越しくださった時はぜひ我らコンピューター部に
投票して頂けると嬉しいです。
